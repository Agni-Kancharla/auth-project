# Authentication System

## Features

* User Registration
* User Login
* Password Hashing
* Form Validation
* Show/Hide Password
* Responsive Design
* Bootstrap 5 UI
* PHP & MySQL Backend

## Technologies Used

* HTML
* CSS
* JavaScript
* Bootstrap 5
* PHP
* MySQL

## Installation

1. Import database.sql into phpMyAdmin.
2. Start Apache and MySQL in XAMPP.
3. Open:
   http://localhost/auth-project/index.html
